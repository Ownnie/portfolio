## Key Contributions

- **License Control Platform**: End-to-end development (backend + frontend) for centralized client-license management, expiration tracking, and automated status enforcement.
- **SuiteCRM Iframe Module**: Real-time guard geolocation on interactive maps, providing supervisors with live field visibility within the CRM.
- **Offline-first architecture**: Local data persistence, configurable module downloads, background synchronization, and automatic retry mechanisms.
- **Facial recognition**: Enrollment and verification integrated into the authentication flow for on-site security personnel.
- **SuiteCRM integration**: Endpoints v4.1 (`login`, `get_entry_list`, `get_module_fields`, etc.), session token management, and `baseUrl` sanitization.
- **Clean Architecture**: Repositories and use cases (domain) decoupled from the framework; unit tests for services.
- **Sync strategies**: Incremental pull/deferred push and timestamp-based conflict resolution.
